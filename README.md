# ⏱️ Ponto Pro

Sistema simples de **controle de ponto diário** com cálculo automático de horas, histórico de registros, gráfico de horas e integração com **Google Sheets**.

O objetivo do projeto é permitir que o usuário registre:

- Entrada
- Saída para almoço
- Volta do almoço
- Saída final

O sistema calcula automaticamente o **saldo de horas do dia** considerando uma jornada padrão de **9 horas (540 minutos)**.

Os dados ficam armazenados em:

- LocalStorage do navegador
- Google Sheets (backup automático)

---

# 📊 Funcionalidades

✔ Registro de ponto diário  
✔ Cálculo automático de horas trabalhadas  
✔ Cálculo de saldo (+ ou -)  
✔ Histórico de registros  
✔ Gráfico de horas trabalhadas  
✔ Exportação em CSV  
✔ Backup automático no Google Sheets
